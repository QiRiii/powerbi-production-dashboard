This project analyzes machine performance using Power BI, focusing on output, downtime, and efficiency.

## 🔍 Key Insights
- Machine M02 underperforms due to high breakdown-related downtime
- Breakdown maintenance is the main contributor to total downtime
- Performance differences may be influenced by machine allocation

## 💡 Tools Used
- Power BI
- Power Query
- DAX
